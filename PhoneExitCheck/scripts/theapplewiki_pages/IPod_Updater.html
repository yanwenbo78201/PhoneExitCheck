iPod Updater (also known by its later internal project name, iPod Updater Extreme) was a utility to update or restore (format/erase) an iPod to the latest firmware.

iPod Updater is released for Windows 2000 and later, in addition to Mac OS X as a PowerPC and Intel 32-bit universal binary. Mac OS 9.2.1 was supported for the first and second generation iPods until at least iPod Software 1.2.2 Updater. It supported Windows Me until iPod Software 1.3 and 2.0.1 Updaters (Windows 98 lacks support for USB/FireWire mass storage devices).

The installation .pkg files use strict checking of the operating system version range, refusing to run on versions later than it is intended for. The final release, iPod Updater 2006-06-28, supports up to Mac OS X 10.4.x. To restore an iPod shuffle (1st generation) on Windows, a 32-bit operating system installation is required.

On Macs, Software Update would automatically download new versions of iPod Updater. Subsequently, launching iTunes with an outdated iPod connected would display a prompt requesting to launch iPod Updater.

Initially, the app was only required to support the first and second generation iPod, which use identical software. With the release of iPod with dock connector (3rd generation), and later iPod mini, separate branches of the app were released for each device and firmware. With version 2004-04-28 (3.0), the separate apps were merged into one unified app, named "iPod Updater", followed by the date of release.

On 12 September 2006 (2006-09-12), iTunes 7.0 was released with the ability to update and restore iPods, obsoleting iPod Updater. This release saw the introduction of the IPSW file format, allowing iTunes to download the latest firmware on an as-needed basis. The format was later extended for use with the Apple TV with iTunes 7.1, and iPhone with iTunes 7.3. The iPod Updater backend was moved to iPodUpdater.framework, bundled with iTunes. With macOS Mojave, the framework was moved to PrivateFrameworks in the operating system.

iPod Reset Utility (initially iPod shuffle Reset Utility) was a variant of iPod Updater released with specific support for the iPod shuffle (1st generation), later adding support for the iPod shuffle (2nd generation). It is intended to be used when a first-generation iPod shuffle flashes orange and green LEDs when attempting to play music, or if it is not appearing as a USB removable storage device when connected to a computer.[10]

iPod Reset Utility is released for Windows 2000 SP4 and Windows XP SP2, in addition to macOS as PowerPC and Intel 32-bit universal binaries. Unlike iPod Reset Utility, it is supported on all releases of macOS up to 10.15 Catalina, which removes support for Intel 32-bit applications.

As restoring an iPod shuffle (1st generation) requires a driver (StMp3Rec.sys) that is only provided as a 32-bit version, iPod Reset Utility Setup refuses to install on 64-bit Windows. Restoring these models on 64-bit Windows was not supported until the release of iTunes 7.6. While the driver's .inf file indicates support for Windows 98 and Windows Me, the main iPod Reset Utility application does not run on Windows 9x.

Observations

iPod shuffle Reset Utility 1.0 references the then-announced, but not yet released, iPod shuffle (2nd generation) in Icons.plist, despite a matching icon not being included. Additionally, a Doxygen Doxyfile build script is found inside the app's resources. However, it doesn't appear to contain anything interesting.

In version 1.0.0.53 for Windows, strings for non-English localizations in the main iPodshuffleResetUtility.exe refer to "Apple iPod Q98", the device's codename. It is unclear where these localization strings are used, as the UI strings are actually loaded from language-specific res.dll files, not the main executable.

In versions 1.0.3 and later, all variants of the iPod shuffle (2nd generation) have a shared firmware, labelled as firmware identifier 130 (its later revisions are 131 and 133), while the IPSWs for this iPod are unique to each revision.

In version 1.0.4.71 for Windows, the iPod shuffle (2nd generation) DFU mode image is referred to as "S5L8441 Rev.1" in the embedded FirmwareVersions.plist (IDR_DESC PE resource), while the chip itself is labelled S5L844A. It is therefore believed that the "A" suffix represents a revision to this chip.
